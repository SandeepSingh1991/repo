Login not working?

  1. Create the same email/password in Supabase Auth → Users.
  2. Run supabase-schema.sql in Supabase SQL Editor.
  3. Add the exact same email in users_role, for example admin@gmail.com or user@gmail.com.
  4. Use Google Drive /preview links for videos.

Quick setup

Before login, do these 3 steps in Supabase

1. Auth user banao

Supabase Dashboard → Authentication → Users → Add user. Example: admin@gmail.com.

2. SQL run karo

Repo ka supabase-schema.sql copy karke SQL Editor me run karo.

3. Same email role me dalo

Email mismatch hua to login ke baad role nahi milega. Auth email aur users_role.email same hona chahiye.

insert into public.users_role (email, role)
values
  ('admin@gmail.com', 'admin'),
  ('user@gmail.com', 'user')
on conflict (email) do update set role = excluded.role;
